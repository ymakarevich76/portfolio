Проекты

В каждом проекте весь код разбит по блокам, использованы семантические тэги HTML5. Использована методология БЭМ, препроцессор sass, а HTML собирается с помощью gulp-file-includer. Также производится оптимизация веса картинок. Для анимации некоторых сайтов была использована библиотека animate.css и wow.js. Для слайдера используется Swiper.
